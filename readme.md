# Landing Page - JavaScript

Uma landing page responsiva desenvolvida com HTML5, CSS3 e JavaScript vanilla.

## Preview

![Landing Page Screenshot](src/images/screenshot.png)

## Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos de desenvolvimento Front-End, trabalhando com responsividade, animações, manipulação do DOM e organização de estilos CSS.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- jQuery
- ScrollReveal

## Funcionalidades

- Design responsivo para mobile, tablet e desktop
- Navegação dinâmica com efeitos suaves
- Formulários com interatividade
- Performance otimizada

## Como usar

Clone o repositório:

```bash
git clone https://github.com/RudolfoSehnemCode/LandingPage-Js.git
cd LandingPage-Js
```

Abra o arquivo `index.html` no navegador ou use um servidor local:

```bash
# Com Python 3
python -m http.server 8000

# Com Node.js
npx http-server
```
